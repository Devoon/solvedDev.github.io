# Item-Generator
This generator simplifies the process of adding custom items to Minecraft Bedrock.

## How it works
Download this repository and navigate to the index.html file. Open it in your browser. Now select the file to which you want to add a custom item behavior (logical choice: player.json). The second file you need to select is a file where you have described your new item behavior. To learn the syntax, scroll further down. After you have selected both files, click "Parse" and you get the edited player.json. It is saved into your downloads folder.
